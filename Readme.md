# go-tools

与业务无关的 golang 工具补充

> 有任何需求，请优先考虑 [lo 工具集](https://github.com/samber/lo) 避免重复造轮子

## 使用方法

- 请参考演示项目[zero-vue-demo/api-center](https://github.com/zero-vue-demo/api-center)

## 开发流程

```sh
# 在主项目中执行以下命令
cd work
git clone git@github.com:5-say/go-tools.git
cd ..
go work use ./work/go-tools
code ./work/go-tools
```
